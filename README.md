# 🎸 Riff Buddy

Upload a guitar riff → get your key, scales, chords, and songs to jam with.

Built as a birthday gift for a guitar player who wants to have more fun without the theory homework.

## Features
- **Key Detection** — Upload a recording, get the key automatically (librosa + Krumhansl-Schmuckler algorithm)
- **Scale Maps** — Minor Pentatonic, Major Pentatonic, and Blues Scale with fretboard diagrams
- **Chord Family** — See which chords belong together, with beginner-friendly shapes highlighted
- **Song Suggestions** — Curated jam-along songs with YouTube links

## Deploy on Streamlit Cloud
1. Push this repo to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect your GitHub account
4. Select this repo → `app.py` as the main file
5. Deploy — done

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
